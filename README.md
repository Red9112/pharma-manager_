# PharmaManager

Application de gestion de pharmacie développée dans le cadre du test technique SMARTHOLOL.

## Stack technique

- Backend : Python, Django, Django REST Framework, PostgreSQL, drf-spectacular
- Frontend : React.js (Vite), Axios, React Query (ou équivalent pour la gestion des requêtes)

Ce README sera complété au fur et à mesure de l’avancement (installation backend/frontend, configuration de la base de données, documentation API, etc.).
